Taller de Git y GitHub

Bienvenido a este taller de Git y GitHub

¿Qué es Git y GitHub?

Git es un sistema de control de versiones distribuido que permite hacer versiones de sus proyectos y compartir sus trabajos con otros desarrolladores. GitHub es una plataforma en la nube que permite alojar repositorios de Git y manejar sus proyectos.

En este taller, aprenderás a:

  • Instalar Git y GitHub
  • Crear un repositorio de Git
  • Clonar un repositorio de Git
  • Hacer cambios en un repositorio de Git
  • Commitear cambios en un repositorio de Git
  • Pushear cambios en un repositorio de Git
  • Pullear cambios en un repositorio de Git
  • Compartir un repositorio de Git
  • Forzar un push en un repositorio de Git
  • Eliminar un repositorio de Git

¿Qué necesito para el taller?

Para este taller, necesitarás una computadora con Windows 10 o macOS.

Si no tienes una computadora con Windows 10 o macOS, puedes usar una computadora virtual con Windows 10 o macOS.

También necesitarás una cuenta de GitHub.

Si no tienes una cuenta de GitHub, puedes crear una cuenta gratuita en GitHub.
